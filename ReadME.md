```
1. Create your project directory and initialize Node.js:

mkdir security-quiz
cd security-quiz
npm init -y

2. Install Dependencies
npm install express cors nodemon

express → Backend framework for handling routes.
cors → Allows frontend to make API requests.
nodemon → Auto-restarts the server during development.

3. Create a file named server.js

4. Inside your project folder, create a public folder with index.html, style.css, and script.js.

public/index.html
```
